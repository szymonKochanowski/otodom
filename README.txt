Project created to use and learn about threads in application.

Application name is otodom.

The main aim of application is upload list of offers with apartments to sale for specific city, with specified
price between for square meter. Default limit of offers is set for 100 plus 3 promoted offers.
City name can not contain capital letter and without polish marks.
